What it is

Ensemble is a framework for building RL environments where several simulated users and several agents share one typed world. You describe the world in Rust, you write the scenarios in Python, and the framework handles message routing, tool dispatch, the trace log, and the grader that turns each rollout into a reward.

Most agent benchmarks model one agent talking to one human in a stateless sandbox. That is not how production support, sales, customer success, or operations actually work. Ensemble lets you build the multi-actor version, run rollouts at scale, and turn the rollouts into training signal.

Install

uv sync                       # build the python extension, install the workspace
cargo build -p ensemble-cli   # produces ./target/debug/ensemble

The docs page has the full quickstart, with a walkthrough of how to write a world, a scenario, and a grader, and how to use the framework as an RL environment.
