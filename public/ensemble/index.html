What it is

Ensemble is a framework for building RL environments where multiple simulated users and multiple agents share a single typed world. You write the world in Rust, the scenarios in Python, and the framework handles message routing, tool dispatch, the trace log, and the grader. Plank is the worked example: a small fake SaaS with SQLite-backed state, six tools, and five personas.

The framework is small on purpose. The core scheduler is one tokio task per actor and an event log; the LLM layer is a single LLMBackend trait with Anthropic, OpenAI, vLLM, and mock implementations; the Python sdk is one @scenario decorator and a World handle.

Install

uv sync
cargo build -p ensemble-cli

uv sync builds the native extension via maturin and installs the Python sdk, the Plank example, and the training package. cargo build produces the ensemble CLI under ./target/debug/.

The docs page has a longer quickstart and walks through each part of the architecture.
