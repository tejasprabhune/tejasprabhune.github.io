ensemble

Multi-user, multi-agent RL environments.

Overview Docs Scenarios Plank Personas

Two modes

Prompted personas live entirely in their system prompt. A frontier model plus a hidden-state block is enough for most agent-facing tests. Trained personas swap the frontier model for a smaller fine-tuned one (DPO+LoRA on Qwen-2.5-7B by default), served through a vLLM endpoint. The scenarios don't know which is which; both implement the same LLMBackend trait.

Spec format

[persona]
name = "frustrated_power_user"
mode = "trained"
description = "Long-tenured paid user with low patience."

[persona.style]
tone      = "frustrated"
verbosity = "high"

[persona.demographics]
plan          = "team"
tenure_months = 36

[persona.hidden_state.schema]
mood        = { type = "string", default = "annoyed" }
hidden_goal = { type = "string", default = "refund_3mo" }

[persona.system_prompt]
template = """
You are a long-tenured Plank customer. You want a refund...
"""

[persona.training]
base_model   = "Qwen/Qwen2.5-7B-Instruct"
backend      = "modal"
dataset      = "spec_only"
adapter_name = "ensemble-plank-frustrated-power-user"

[persona.training.lora]
r       = 16
alpha   = 32
dropout = 0.05

[persona.training.dpo]
beta          = 0.1
epochs        = 1
learning_rate = 5e-6

Training a persona

uv pip install 'ensemble-train[torch]'
./target/debug/ensemble train \
  examples/plank/personas/frustrated_power_user.toml --backend modal

When dataset = "spec_only" the pipeline runs a self-play loop to build a DPO preference dataset before training. The breaker is a frontier model that tries to push the persona out of character; in-character continuations become chosen, breaks become rejected.

The trainer (TRL DPOTrainer on a LoRA adapter) writes to checkpoints/<persona>/ and emits a persona-card.md summarizing the run. If $HF_USERNAME is set, the adapter is pushed to the hub.

Consistency benchmark

Persona consistency is evaluated by an adversarial classifier sweep. The breaker model gets N turns to push the persona; the score is the fraction of turns the persona stayed in character.
